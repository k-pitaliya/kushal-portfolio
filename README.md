Personal portfolio of **Kushal Pitaliya** — VLSI Design Verification Engineer & Cloud Architect.  
ECE student at CHARUSAT building at the intersection of silicon and the cloud.

## Stack

- **Framework:** Next.js 15 (App Router, Turbopack)
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion, Three.js
- **Deployment:** Vercel

## Highlights

- SystemVerilog · UVM 1.2 · RTL Design · FPGA Prototyping (Spartan-6)
- AWS · Serverless · Terraform · Docker · Kubernetes
- Embedded C · STM32 · AVR · RTOS

## Run locally

```bash
npm install
npm run dev
```
