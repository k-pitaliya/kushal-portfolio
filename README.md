Personal portfolio of **Kushal Pitaliya** — VLSI Design Verification Engineer who also ships cloud and embedded systems.  
ECE student at CHARUSAT building at the intersection of silicon and software.

## Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS 4
- **Type:** Fraunces (display) · Geist (body) · JetBrains Mono (technical)
- **Animations:** Framer Motion · Lenis (smooth scroll)
- **Deployment:** Vercel

## Highlights

- SystemVerilog · UVM 1.2 · SVA · RTL Design · FPGA Prototyping (Spartan-6)
- AWS (Lambda, API Gateway, DynamoDB) · Serverless (SAM/CDK) · Terraform · CI/CD
- Embedded C · STM32 · AVR · CMSIS-DSP

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
